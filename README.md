# Análisis de Mercado: La Gran Prueba de Sabor (Coffee Survey)

## Resumen del Proyecto
Este proyecto actúa como una consultoría técnica para un grupo inversor interesado en abrir una cafetería en Estados Unidos. El objetivo es analizar las preferencias de más de 4,000 consumidores para determinar el target ideal, métodos de preparación favoritos y hábitos de consumo.

## Principales Insights
* **Target Demográfico:** El grupo de 25-34 años representa el mayor volumen de consumidores.
* **Hábitos:** El consumo en casa es predominante, seguido por la oficina.
* **Preferencia:** (Aquí añade un insight que hayas descubierto, ej: El 'Pour over' es el método preferido en X grupo).

## Herramientas Utilizadas
- **Python / Pandas:** Limpieza de datos y manipulación de columnas con opciones múltiples (método `explode`).
- **Matplotlib:** Generación de gráficos de torta y barras para distribución etaria y preferencias.
- **Google Colab:** Entorno de desarrollo.

## Cómo ejecutarlo
1. Clona el repositorio: `git clone https://github.com/tu-usuario/coffee-consumer-insights.git`
2. Instala las dependencias: `pip install pandas matplotlib requests`
3. Abre el archivo `notebooks/analisis_cafeteria.ipynb` en Jupyter o VS Code.

## Estructura del proyecto
- `notebooks/`: Contiene el análisis paso a paso.
- `reports/figures/`: Gráficos exportados del análisis.
